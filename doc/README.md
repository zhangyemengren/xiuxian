## 只记三条规则

1. 开发时只看当前步骤的一个文档。
2. 中途想到任何新内容，先追加到 [ideas.md](./ideas.md)，不要立即修改当前步骤。
3. 新想法确定现在实现后，直接追加到当前步骤文档的对应标题下。

## 项目方向

这是一个可以长期演化的修仙模拟。第一阶段先做命令行自动模拟，确保整个过程可运行、可观察、可复现。角色、修炼、事件、战斗和成长会逐步加入；玩家交互、地图、宗门等内容以后再做。
