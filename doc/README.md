## 只记三条规则

1. 开发时只看“当前任务”，不用提前研究全部文档。
2. 中途想到任何新内容，先追加到 [ideas.md](./ideas.md)，不要立即改当前任务。
3. 新想法需要现在实现时，在当前步骤的任务表中增加一项；已有任务不用改名。

## 项目方向

这是一个可以长期演化的修仙模拟。第一阶段先做命令行自动模拟，确保整个过程可运行、可观察、可复现。角色、修炼、事件、战斗和成长会逐步加入；玩家交互、地图、宗门等内容以后再做。
