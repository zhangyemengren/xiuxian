# 想法与未来功能

任何中途想到的内容都先追加到本文件。记录想法不代表现在就要实现。

## 刚想到的内容

- 第一阶段希望尽可能完整地记录未来设定和功能，但实现仍需逐步推进。

## 模拟工具

- 随机种子和结果复现；
- 暂停、继续、倍速和单步执行；
- 多次批量模拟与结果对比；
- 规则配置文件和配置校验；
- 存档、读档和版本兼容；
- 调试模式与指定事件注入；
- 日志级别和状态快照间隔。

## 角色与成长

- 基础属性、生命、灵力、经验和资源；
- 灵根、天赋、气运、寿元和心境；
- 境界、等级、突破与瓶颈；
- 功法、技能、熟练度和流派；
- 装备、法宝、器灵和套装；
- 伤势、毒、诅咒和永久缺陷；
- 死亡、转世、夺舍和传承。

## 修炼与生产

- 多种修炼方式及不同消耗和收益；
- 洞府、灵脉、聚灵阵和闭关；
- 炼丹、丹方、药材和丹毒；
- 炼器、材料、品质和耐久；
- 制符、阵法、傀儡和御兽；
- 采集、种植、挖矿和生产链。

## 事件、探索与战斗

- 随机事件、宝物和修炼失败；
- 基础战斗与多回合战斗；
- 技能、状态效果和逃跑；
- 地图、地点、移动和区域危险度；
- 秘境、副本、遗迹和事件链；
- 怪物生态、掉落表和图鉴；
- 天劫、心魔和突破事件。

## 世界与社会

- 世界时间、季节和纪年；
- 宗门、职位、贡献和宗门任务；
- 城镇、商店、拍卖和经济波动；
- NPC、关系、好感、仇恨和因果；
- 家族、师徒、道侣和后代；
- 势力冲突、战争和世界事件；
- 声望、善恶、通缉和阵营。

## 展示与扩展

- 可读时间线和模拟汇总统计；
- 文本交互界面；
- 图形界面和数据图表；
- 成就、里程碑和历史档案；
- 模组、自定义规则和内容包；
- 自动平衡分析和异常检测。

## 候选配置

这些配置只作为提醒，进入具体任务时再确定字段名称、类型和默认值。

| 分类 | 可能需要的配置 |
| --- | --- |
| 模拟 | 回合数、随机种子、日志级别、快照间隔、死亡后是否停止、批量次数 |
| 角色 | 初始 HP/MP、属性范围、等级、经验、资源、寿元、气运和天赋 |
| 修炼 | 修炼类型、灵力消耗、经验收益、选择权重、属性要求、恢复速度 |
| 事件 | 事件权重、宝物范围、失败损失、冷却时间、事件链长度 |
| 战斗 | 攻击波动、怪物等级、伤害、奖励、惩罚、回合上限、逃跑率 |
| 成长 | 升级经验、属性成长、成长权重、等级上限、突破条件和成功率 |
| 世界 | 纪年、季节、价格、地点危险度和势力关系 |

## 处理想法

- 不确定什么时候做：继续留在这里。
- 决定现在做：在对应步骤的任务表增加一项。
- 决定以后做：留在这里，等规划后续阶段时再取用。
- 决定不做：直接删除或划掉并写一句原因。
